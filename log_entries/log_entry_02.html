VSCode Magic

13 Dec 2023

I have been trying to setup a workflow for myself in order to document my shenanigans here as I go. The concept was simple, put thoughts and processes on site. The execution was not so simple, go back in high school and write my them in css and html (this isn't wordpress yall). I've been doing this in trusty old VSCode, but have been pushing my commits to Github so that I can see if the page formatting was right, etc, etc. This got old really fast, and I was considering setting up a server when I thought that VSCode probably has an extension for this. Unlike myself, it did not disappoint. I got me the Live Preview extension and can now check the pages I'm working on without having to push commits like a maniac.
